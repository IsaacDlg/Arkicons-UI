# Arkicons UI Fragment

Extracto de componentes UI desarrollados para un cliente del sector construcción.
Este repositorio demo se enfoca en la implementación de la navegación interactiva y la estructura de la página de inicio.

## Stack
- React
- Vite
- CSS3 (Variables, Grid, Flexbox)

## Estructura
- `/components/CardNav`: Sistema de navegación con tarjetas expandibles y gradientes.
- `/pages/Home`: Landing page con animaciones de entrada (Intersection Observer) y grid responsive.
